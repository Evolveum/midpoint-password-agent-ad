# Installation of Active Directory

Prerequisities:
- Installed Windows Server


Use Server Manager application in Windows Server to setup AD.

1. Open Server Manager -> Dashboard
2. Click `Add roles and features`
3. Next until in Installation Type step
4. Installation Type = `Role-based or feature-based installation`
5. Server selection - select that one server that is available
6. Server Roles - **Important** - Choose `Active Directory Domain Services` and `DNS Server`
7. Features - no need to pick anything
8. Keep clicking next until Installation is available
9. Click Install
